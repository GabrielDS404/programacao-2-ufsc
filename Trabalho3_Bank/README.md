# Sistema Bancário com Interface Gráfica (C++ e Qt)

Projeto prático final desenvolvido para a disciplina de Programação 2 (Engenharia de Computação - UFSC).

## 📌 Sobre o Projeto
Trata-se de uma aplicação desktop completa para gerenciamento bancário. O diferencial deste projeto é a utilização do **Qt Framework** para a construção de uma Interface Gráfica do Usuário (GUI), saindo das aplicações padrão de terminal e oferecendo uma navegação intuitiva por janelas e menus.

## ⚙️ Funcionalidades Principais
* **Gestão de Clientes:** Cadastro, listagem, busca por CPF e exclusão de clientes.
* **Gestão de Contas:** Criação de Contas Correntes e Contas Especiais (com limite), vinculadas aos clientes.
* **Operações Bancárias:** Interface dedicada para realizar Depósitos, Saques e Transferências entre contas, com validação de saldo e limites.
* **Arquitetura:** Forte aplicação de Orientação a Objetos (Herança e Polimorfismo) para separar as regras de negócio de cada tipo de conta bancária.

## 🛠️ Tecnologias
* C++ 
* Qt Framework / Qt Creator
