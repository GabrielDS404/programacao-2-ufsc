# Gerenciador de Strings em C++

Projeto prático desenvolvido para a disciplina de Programação 2 do curso de Engenharia de Computação (UFSC).

## 📌 Sobre o Projeto
Trata-se de uma aplicação interativa via terminal que permite armazenar, buscar e remover palavras (strings) de uma lista. O programa salva os dados automaticamente em um arquivo de texto (`texto.txt`), garantindo que nenhuma informação seja perdida ao fechar o sistema.

## ⚙️ Funcionalidades Principais
* Inserção e visualização de palavras cadastradas.
* Busca por palavras exatas ou trechos (substrings).
* Remoção de itens pelo índice ou por correspondência de texto.
* Leitura e gravação automática em arquivo de texto.
